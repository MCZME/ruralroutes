# Rural Routes / 乡野商路

**A village-centered item trading system for Minecraft.**

<p align="center">
  <a href="README.md">中文</a> | <a href="README-EN.md">English</a>
</p>

乡野商路是一个以村庄为核心的物品交易模组。它为村庄加入独立的贸易站、展示柜、传闻板和市场波动，让不同村庄之间形成真正可区分的交易关系。它不是原版村民交易的替代品，而是另一套围绕村庄展开的交易方式。

如果你喜欢村庄差异、物品交换和轻量经济系统，这个模组会让每个村庄都有自己的交易价值。

目前模组仍处于测试版本，内容和玩法细节后续可能继续调整。如果你对内容设计或玩法方向有建议，欢迎到 GitHub Issues 提出。

## 核心玩法

在原版中，资源通常来自固定基地、农场、挖矿和等待。乡野商路提供了另一种节奏：

1. 发现一个村庄。
2. 查看贸易站、展示柜和传闻板。
3. 判断这个村庄正在出售什么、收购什么、市场有什么变化。
4. 用背包里的物资换取你需要的商品或其他交易机会。
5. 当库存用尽、收购饱和或价格不合适时，继续前往下一处村庄。

模组的重点是把村庄之间的交易差异做成一套可以长期使用的系统。

## 主要内容

### 贸易站

贸易站是模组的主要交互入口。每个被激活的村庄贸易站都会拥有自己的库存、收购方向和交易周期状态。

你可以在这里：

- 从村庄购买物资
- 向村庄出售它需要的物品

### 村庄主题

不同群系的村庄会有不同的商品倾向与交易侧重。平原、沙漠、热带草原、针叶林和雪地村庄不会只是换一套外观，而会拥有不同的资源身份。

有些村庄更适合获取常见材料，有些更偏向石材、染料、木材、皮革、冰雪资源或加工品。你需要根据所在地区和当前需求决定是否在这里交易。

### 展示柜

展示柜用于快速观察一个村庄当前正在出售的代表性物品。

它让玩家在进入完整交易界面之前，就能大致判断这个村庄是否值得停留，或者是否更适合作为之后路线中的补给点。

### 传闻板

传闻板会根据当前市场状态生成模糊情报。

它不会直接告诉你精确数值，而是用更自然的方式提示哪些货物近期紧俏、哪些地方可能有更多供应、哪些主题或区域正在发生变化。传闻板的目标不是替你做决定，而是给旅途一个方向感。

### 市场波动

每个贸易周期都会生成新的市场事件，影响部分物品的价格与库存。

这些变化可能是全局的，也可能只影响某些群系或村庄主题。它们会让不同村庄在不同时间拥有不同价值，鼓励玩家观察、记录和移动，而不是永远停在同一个地方反复交易。

## 数据包驱动

本模组的绝大多数内容都可以通过数据包设置，你可以完全自定义其内容。

## 这个模组适合你吗？

如果你喜欢这些体验，乡野商路会很合拍：

- 你希望村庄之间的交易关系更有差异。
- 你喜欢围绕物品供需做取舍。
- 你想要一套不依赖原版村民列表的交易系统。
- 你喜欢轻量交易系统，但不想被复杂模拟压住。

如果你只想要一个快速获取全部资源的商店模组，乡野商路可能不是那个方向。它更关注村庄、交易和市场变化。

## 如何开始

进入任意村庄并寻找这些方块：

| 方块 | 作用 |
| :--- | :--- |
| 贸易站 | 打开主要交易界面 |
| 展示柜 | 查看村庄代表性出售物品 |
| 传闻板 | 查看当前周期的市场传闻 |

建议第一次游玩时先做三件事：

1. 找到一个村庄贸易站。
2. 看看它出售什么、收购什么。
3. 尝试用附近能获得的物品完成一次交换。

完成第一笔交易后，模组的节奏会自然展开：你会开始思考哪些东西值得带走，哪些村庄值得回访，下一段路该往哪里走。

## FAQ

**这个模组会替换原版村民交易吗？**

不会。乡野商路使用独立的贸易站系统，不替换原版村民交易。

**必须使用这个系统才能正常生存吗？**

不必须。它提供一条额外资源路线，你可以按需使用。

**这个模组更偏向什么体验？**

它更偏向村庄之间的物品交换和市场变化，而不是单纯的村民商店。

**为什么有些村庄库存不够？**

这是模组的核心约束之一。库存有限会让玩家自然流动，而不是永远停在同一个地点。

**传闻板为什么不显示精确数字？**

传闻板提供的是方向感，而不是计算器。它帮助你判断哪里可能值得去，但最终路线仍由玩家自己决定。
