以"获取难度"为基准，综合四个维度计算物品基础价值。

## 计算维度

### 一、获取难度

| 难度等级 | 说明 | 系数 | 示例 |
| :--- | :--- | :--- | :--- |
| 1 | 手可采，无限可再生 | 1-2 | 泥土、圆石、沙子、原木 |
| 2 | 需工具采集或简单加工 | 3-8 | 木板、木棍、燧石、羽毛 |
| 3 | 需熔炼或合成 | 10-25 | 铁锭、玻璃、面包、陶瓦 |
| 4 | 稀有矿或特定群系产物 | 30-60 | 钻石、金锭、黏液球、冰 |
| 5 | 下界/末地维度物品 | 60-120 | 烈焰棒、荧石、末影珍珠、潜影壳 |
| 6 | 极稀有/不可再生 | 120+ | 下界合金锭、附魔金苹果、鞘翅 |

### 二、可再生性

| 类型 | 说明 | 修正 |
| :--- | :--- | :--- |
| 无限可再生 | 可通过农场/刷怪塔量产 | ×0.8 |
| 有限可再生 | 需消耗不可再生资源来生产 | ×1.0 |
| 不可再生 | 开采后不恢复 | ×1.3 |

### 三、加工链路长度

| 加工层级 | 说明 | 附加值 |
| :--- | :--- | :--- |
| 原始 | 直接采集 | +0% |
| 初级加工 | 木板、木棍、熔炼产物 | +30% 基础值 |
| 中级加工 | 面包、箱子、铁轨 | +60% 基础值 |
| 高级加工 | 蛋糕、附魔台、信标 | +100% 基础值 |

### 四、稀有度参数

物品在原版中的标准生成频率，作为微调因子（±20%）。

## 计算公式

```
物品基础价值 = [难度系数基准值 × 加工链路附加值] × 可再生性修正 × 稀有度微调
```

最终价值取整数，便于显示和计算。

### 默认价值

未匹配到任何价值规则的物品，使用配置文件中的默认价值 `values.default = 1`。

### 价值解析优先级

1. 单品覆写（如 `minecraft:iron_ingot = 12`）—— 最高优先级
2. 物品池标签基线（如 `#ruralroutes:pool/mineral = 8`）—— 次优先级
3. 配置默认值（`values.default = 1`）—— 兜底

## 为什么不用原版村民交易价

原版交易存在大量"玩家福利"定价（如制箭师换木棍给绿宝石），缺乏内在逻辑。

本模组需要一套更有内在逻辑的价值表，让交换感更公平。
