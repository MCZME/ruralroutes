## 决策

村庄通过建筑风格和视觉设计体现主题辨识度。

## 具体设计

| 项目 | 决策 |
| :--- | :--- |
| 建筑体量 | 中型带特征，每个村庄一个 |
| 建筑风格 | 按群系分模板，主题改材质和少量装饰 |
| 建筑位置 | 融入村庄生成逻辑，具体位置由生成逻辑处理 |
| NPC交互 | 不做NPC对话 |

## 设计意图

### 发现感

玩家走进村庄能感受到「这个村和上一个不一样」。

### 视觉辨识

建筑风格随群系变化，主题通过材质和装饰微调。玩家看到建筑就能大致判断村庄主题。

### 简洁交互

不引入NPC对话系统，所有交互通过方块完成，保持原版风格。
