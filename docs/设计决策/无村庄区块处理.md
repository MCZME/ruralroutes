# 无村庄区块处理

## 问题

贸易站首次交互时，如果检测到旁边没有村庄，应该如何处理？

## 决策

**直接跳过，无响应。**

**Why:**

贸易站是自然生成、与村庄绑定的建筑。无村庄的贸易站属于异常情况（可能是玩家手动放置、结构模板错误等），不应提供正常功能。

**How to apply:**

- 检测使用 `PoiManager.sectionsToVillage()` 判断是否在村庄范围内
- 无村庄时返回 `InteractionResult.PASS`，不执行任何操作

## 影响范围

- 贸易站初始化流程
- 商业节点创建条件
