## 决策

村庄交换通过三个核心方块实现：

- **贸易站**：承载常规买卖
- **展示柜**：陈列特产和特殊交换规则
- **传闻板**：显示当前刷新周期的模糊价格情报

## 原因

### 分离功能，各司其职

| 方块 | 功能 |
| :--- | :--- |
| 贸易站 | 处理所有村庄通用的买卖 |
| 展示柜 | 突出每个村庄的独特性（特产名片） |
| 传闻板 | 提供全局市场信息，驱动玩家移动 |

### 视觉辨识

展示柜作为「村庄名片」，传闻板显示简略图标（是否有新消息、消息种类）。

### 界面简洁

贸易站界面分「出售」「收购」两个标签页。
展示柜用独立小界面，只列特产和特殊规则。
传闻板右键打开查看详细情报。

## 具体设计

| 项目 | 决策 |
| :--- | :--- |
| 贸易站界面 | 出售/收购分标签页 |
| 展示柜界面 | 独立小界面，只列特产和特殊交换规则 |
| 传闻板界面 | 右键查看详细情报，表面显示简略图标 |
| 方块信息呈现 | 仅交互入口，无表面状态显示 |
| 命名与材质 | 统一名称，材质随群系换色 |
