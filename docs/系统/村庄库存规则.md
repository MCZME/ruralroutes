# 村庄库存规则

单个村庄的库存上限、消耗方式、买卖库存关系。

## 库存上限

由主题模板定义。

设计主题时，为每种物品直接指定该主题下的库存基准值。

## 共享库存池

村庄对每种物品有一个**净持有量**，买卖共享同一库存池。

| 行为 | 库存变化 |
| :--- | :--- |
| 玩家购买 | 库存减少 |
| 玩家出售 | 库存增加（不超过上限） |

- 库存为 0 时，该物品不可购买。
- 库存达到上限时，该物品不可再向村庄出售。

## 收购限制

由共享池上限自然限制。

收购上限 = 库存上限 - 当前库存。

无须额外配额系统。

## 衍生规则

### 出售物品的归属

玩家卖给村庄的物品直接进入该物品的库存池：

- 若物品不在村庄出售清单上，玩家卖入后库存不为零，但无法买回。
- 若物品在出售清单上，玩家可卖完再买回（价格差可能不划算）。

### 刷货币防护

同一村庄对同一物品：**收购价 < 出售价**。

待价值修正系数设计时落实。

### 刷新行为

全局刷新时，库存恢复至基准值。

周期内的买卖行为不跨周期保留。

## 规则总览

| 规则项 | 决策 |
| :--- | :--- |
| 库存上限 | 由主题模板定义 |
| 库存池 | 买卖共享，单一净持有量 |
| 收购限制 | 由共享池上限自然限制 |
| 出售限制 | 库存为零时不可购买 |
| 刷货币防护 | 同村同物品收购价 < 出售价 |
| 刷新行为 | 全局刷新恢复至基准，周期内行为不跨周期保留 |
