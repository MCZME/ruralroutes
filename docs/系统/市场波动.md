# 市场波动

全局统一的市场价格波动系统，在每次全局刷新时触发，创造持续变化的价值差异窗口。

## 定义

市场波动在每次全局刷新时触发，对特定物品池的价格进行临时调整，并通过传闻板向玩家传递模糊情报。

## 触发时机

与全局刷新同时进行。

## 波动结构

每次全局刷新生成两个波动：

| 波动 | 作用域 | 数量 | 影响范围 |
| :--- | :--- | :--- | :--- |
| 波动A | 全局 | 1 | 全图所有村庄 |
| 波动B | 区域 | 1 | 指定群系内的村庄 |

两个波动定义统一：各选中一个物品池，各确定涨跌方向与幅度，仅作用域不同。

市场事件规则支持三种作用域类型：

| 作用域 | 说明 | 匹配方式 |
| :--- | :--- | :--- |
| GLOBAL | 全局生效 | 影响所有村庄 |
| BIOME | 指定群系生效 | 仅影响匹配群系的村庄 |
| THEME | 指定主题生效 | 仅影响匹配主题的村庄 |

THEME 作用域在生成阶段展开为每个主题的独立事件，在解析阶段通过主题 ID 精确匹配。

## 波动内容

| 参数 | 说明 |
| :--- | :--- |
| 目标物品池 | 从所有物品池中随机选取 |
| 方向 | 上涨或下跌，各 50% 概率 |
| 幅度 | 在 `volatility_min` ~ `volatility_max` 之间随机 |

## 物品受影响规则

基于物品的多池归属：

- 物品属于被选中池 → 受该波动影响
- 物品不属于被选中池 → 不受该波动影响
- 物品同时属于多个被选中池 → 叠加

**叠加公式**：

> 最终波动因子 = 1 + Σ(所有命中波动的幅度)

**叠加上限**：总幅度不超过 `market.max_delta`（默认 ±50%）。

## 波动重置

- 每次全局刷新时，上一周期波动完全撤销
- 所有价格修正系数回到主题原始值
- 然后生成新一周期的波动
- 波动不跨周期累积，不永久改变价值表

## 配置项

| 配置项 | 说明 | 默认值 |
| :--- | :--- | :--- |
| `enable_market_volatility` | 是否启用 | true |
| `volatility_min` | 单次波动最小幅度 | 0.15 |
| `volatility_max` | 单次波动最大幅度 | 0.25 |
| `market.max_delta` | 市场事件叠加后的最大价格调整百分比 | 0.50 |
| `global_volatility_count` | 全局作用域波动数量 | 1 |
| `regional_volatility_count` | 区域作用域波动数量 | 1 |

## 与传闻板联动

波动生成后，传闻板显示两层情报：

- **全局情报**：描述全局波动的物品池与涨跌方向
- **区域情报**：描述区域波动的群系、物品池与涨跌方向

情报使用模糊语言，不暴露具体数值。
