## 定义

物品池是对可交换物品进行逻辑分组的工具。它并非游戏内可见的概念，而是驱动多个系统模块的底层组织结构。

## 核心规则

### 多池归属

一个物品可以同时属于多个物品池。归属在配置文件中以列表形式定义。

例如：木炭可同时属于 `[燃料, 木材]`。

### 价格修正取第一池

主题模板对物品的出售/收购修正系数，只看该物品所属物品池列表中的**第一个池**。

这保证了价格修正规则无歧义。

### 市场波动叠加

当市场波动发生时，物品对所有命中的池的波动幅度进行叠加：

> 总波动因子 = 1 + Σ(各命中波动的幅度)

叠加后的总幅度受 `max_total_volatility` 上限保护，防止极端价格。

## 在模组中的角色

| 角色 | 说明 |
| :--- | :--- |
| 价值表组织 | 同一池内物品遵循相近的基础价值计算逻辑 |
| 主题价格修正 | 主题通过物品池（列表第一项）批量定义系数 |
| 市场波动选取 | 每次刷新以物品池为单位随机选取影响目标 |
| 情报文本生成 | 传闻板根据受影响的池生成模糊价格情报 |
