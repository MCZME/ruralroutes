本方案用于重新定义“群系”和“主题”在内容设计中的分工。它服从 [末影龙通关资源链方案](末影龙通关资源链方案.md) 与 [价值与库存设计](../待办/价值与库存设计.md)，先保证“仅靠商路可通关”的主线成立，再细化各群系与主题内容。

## 设计摘要

当前核心判断：

- 乡野商路不是“玩家采集资源，再和交易站换其他资源”。
- 初期采集只负责启动资金，核心玩法应转向跨村庄跑商。
- 群系层负责稳定流通底色，主题层只负责偏移和记忆点。
- 主题不是独立商店，必须成为商路网络中的节点。
- 合约和事件都应服务路线，而不是变成另一套固定村民交易。

玩家主循环应是：

1. 用启动物资换到第一笔货币。
2. 在一个村庄买入低价流出货。
3. 运到另一个高需求村庄卖出。
4. 用利润买入当地下一段路线货。
5. 继续跑下一段商路，而不是回到采集资源重新开始。

## 分层结构

| 层级 | 职责 | 回答的问题 |
|---|---|---|
| 群系层 | 定义稳定流通底色 | 这个群系普遍卖什么、普遍缺什么 |
| 主题层 | 定义主题偏移与身份 | 同群系不同村更偏向哪类货、缺哪类输入 |
| 村庄层 | 承接随机波动与实际生成 | 这一次这个村具体刷出了什么、遇到什么行情 |

当前停用旧命名：

- `主出口`
- `副出口`

当前保留工作名：

- `群系常备供给`
- `群系常备需求`
- `出售偏移`
- `收购偏移`
- `额外货物`

## 群系层规则

群系层定义区域共同点，而不是主题个性。

### 群系常备供给

指大多数同群系村庄都较稳定提供的货物。

作用：

- 建立玩家对群系的基础预期。
- 让同群系多个主题像同一片经济区。
- 给旅行补给和短线交易提供稳定锚点。

### 群系常备需求

指大多数同群系村庄长期愿意接收的货物。

规则：

- 不必与常备供给对称。
- 通常应比常备供给更窄、更明确。
- 核心不是“这里也收什么”，而是“这里普遍缺什么”。
- 组成上可包含少量本地基础资源回收，但主要应表达外来缺口。

### 内容分类

| 分类 | 含义 | 用途 |
|---|---|---|
| 基础资源 | 资源方块、简单食物、低加工本地货 | 建立区域底色 |
| 加工品 | 基础工具、基础合成物、基础加工方块 | 表达区域常见二级流通物 |
| 群系记忆点货 | 玩法设计概念，不等于技术字段 | 强化群系共同记忆点 |

### 需求缺口类型

| 缺口类型 | 当前理解 | 代表方向 |
|---|---|---|
| 生存轴 | 让玩家或村庄待得住 | 口粮、木头、皮革、羊毛 |
| 能源轴 | 支撑烹饪、烧制和加工 | 原木、木炭、煤 |
| 工业轴 | 提升生产能力与耐用品层级 | 粗矿、金属锭、工具、红石 |
| 建造轴 | 提升建筑、烧制和装饰配套 | 石材、黏土、玻璃、装饰建材 |
| 环境轴 | 极端地区额外依赖的驻留物资 | 水桶、寒地补给、远行储备物 |

## 主题层规则

主题层不从零定义一整套买卖表，而是在群系底色上做偏移。

| 字段 | 职责 |
|---|---|
| 核心身份 | 定义这个主题是什么地方 |
| 出售偏移 | 在群系常备供给上强化哪几类货物 |
| 收购偏移 | 在群系常备需求上强化哪几类需求 |
| 额外货物 | 群系基底之外的少量主题个性 |
| 代表货 | 主题卖品中的记忆点候选，不单独占用技术字段 |
| 主题事件 | 适合承接的行情波动 |
| 主题合约 | 可选的机会型交换 |
| 循环定位 | 上游、下游和中转职责 |

这里的“代表货”是玩法设计口径，不表示独立生成链。实际实现中，它们仍然只是当期出售商品的一部分，用来表达“这个主题最像什么”。

主题合约规则：

- 不是必选项。
- 不应为了格式完整硬塞。
- 更适合在当期货流满足条件时，作为加工、打包、整备或地方服务出现。
- 至少一个输入最好来自外部路线，或代表外来补给。
- 输出最好能成为下一段路线的高价值货。

事件规则：

- `release` 类事件应让某地短期适合买入。
- `shortage` 类事件应让某地短期适合运入补给。
- `demand` 类事件应让某地短期适合高价卖出。
- 稀有事件应制造“绕路也值得”的机会。

## 商路循环草案

这些路线是后续判断主题和合约的上位参考。

### 补给换寒地货线

`平原补给 -> 积雪平原 -> 冰雪货 -> 沙漠 -> 染料 / 建材 -> 热带草原或平原`

核心含义：

- 玩家不是去雪地挖冰卖钱。
- 玩家把腹地补给运到寒地终端，再换出寒地特货。
- 冰雪货再流入沙漠绿洲、染坊或其他干旱需求节点。

### 工具换生产线

`平原工坊 -> 针叶林 / 积雪平原 -> 林产 / 猎获副产物 -> 平原或沙漠`

核心含义：

- 工具不只是玩家自用物，也可以成为生产投入。
- 冷地林业、毛皮、猎户主题通过外来工具提高产出。
- 骨块、木炭、毛皮等货再回流到粮仓、染坊或补给节点。

### 燃料与窑业线

`针叶林 / 热带草原木炭 -> 沙漠染坊 / 热带草原陶瓦 -> 装饰建材 -> 平原或建造节点`

核心含义：

- 燃料不是低级杂货，而是加工主题继续运转的商路血液。
- 木炭、染料、陶瓦、带釉陶瓦、砂岩和玻璃应形成连续加工链。

## 价值与库存口径

具体价格和库存不在本文件展开，统一交给 [价值与库存设计](../待办/价值与库存设计.md) 维护。

当前只保留上位口径：

- T1-T4 是赚钱层。
- T5 是花钱层。
- 基础补给高库存、低利润。
- 跑商基础货中库存、中利润。
- 地方代表货和高价值路线货库存更低、利润更高。
- 终局进口货极低库存、极高价格，不负责倒卖赚钱。

## 五个样板群系

| 群系 | 当前经济角色 | 主缺口 |
|---|---|---|
| 平原 | 定居腹地、稳定民生区、商路启动区 | 工业轴 |
| 沙漠 | 干旱建材区、染装加工区 | 能源轴 / 环境轴 |
| 热带草原 | 边地过渡区、粗放生产区、手工业中转区 | 建造轴 / 配套轴 |
| 针叶林 | 能生产的冷地、林地采集区 | 更高阶工业输入 |
| 积雪平原 | 要被补给的冷地、远途停靠区 | 生存轴 |

## 末影龙资源链

终局关键材料不进入群系稳定流通底色，也不应只变成后期刷商店目标。

当前规则：

- `末影珍珠`、`烈焰棒` 进入全局稀有层和终局进口层。
- 它们不应被某个主题永久独占。
- 它们应叠加在群系基底和主题个性之上，作为玩家跑商成果的终局转化目标。

正式内容至少包含四层货物流：

1. 群系共享的稳定流通层。
2. 主题强化与额外内容层。
3. 全局随机稀有卖品层。
4. 终局进口层。

终局进口层当前包含三种入口：

- 全局稀有货架：所有主题都有小概率出现，负责避免锁档。
- 稀有进口事件：短期制造绕路也值得的路线目标。
- 高价值路线收束合约：用高面额货币和高价值商路货换取终局材料。

这里需要守住：

- 不直接出售 `末影之眼`。
- 不让单村或单周期解决完整开门材料。
- 不让后期玩法退化成逐村刷新货架。
- 弓、箭、方块、食物、床材料和基础护甲等战前补给应有稳定贸易来源。

## 防跑偏清单

当前最需要防的是：

- 每个群系单独看都合理，但拼起来没有路线。
- 每个群系都卖一点吃的、卖一点材料、主需求都写成矿物 / 金属。
- 主题层重新长成三套互相独立的商店。
- 玩家记住的是单村菜单，而不是区域路线。
- 合约长期稳定出现，变成另一套固定村民交易。
- 事件只改价格数字，不改变玩家路线选择。

后续补主题或物品池时，优先反查：

- 这个主题低价流出的货要去哪里。
- 这个主题高价流入的货从哪里来。
- 玩家卖完货后，能否买到下一段路线需要的货。
- 合约是在收束路线，还是孤立奖励。
