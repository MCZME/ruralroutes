## 自定义流程

1. 先按 [主题模板](主题模板.md) 创建主题模板，先把主题身份、库存骨架和交易内容引用定下来。
2. 再按 [主题结构](主题结构.md) 制作主题结构，边缘拼图方块先处理好。
3. 在制作结构时，直接到主题结构里的贸易站用模组配置工具右键，并在 UI 中选择第 1 步定好的主题，完成绑定确认。
4. 补齐对应的 [交易内容](交易内容.md) 和配套的 [价值表](价值表.md)；需要行情变化时，再按需补 [标签](标签.md) 和 [市场事件](市场事件.md) 规则。
5. 将主题结构写入对应群系的 `trade_stations.json`，放入 `elements` 列表。
6. 进入游戏检查结构生成、主题绑定，以及传闻板和展示柜是否正常。

## 常见问题

### 为什么对应群系村庄中没有出现我自定义的主题结构？

结构模板中一定要有一个边缘拼图方块，并且方向朝外。可以参考 [主题结构](主题结构.md) 里的图片。

### 为什么我在村庄生成的主题结构中右键贸易站显示数据不匹配？

这是因为在保存结构模板之前，已经右键过贸易站。

解决办法：在主题结构模板中把那个贸易站打掉，重新用配置工具设置，再保存为结构模板。

临时解决方法：直接打掉，再放一个贸易站，然后用配置工具设置。这个方法只能解决当前那一个贸易站，不能解决后面生成的，建议优先用前一种方法。

### 这里的“结构”和“结构模板”分别指什么？

这里的“结构”指游戏内生成出来的建筑。

“结构模板”指用于导出 `.nbt` 文件的建筑本体。

更具体的内容可以查看《我的世界》结构方块的使用说明。
