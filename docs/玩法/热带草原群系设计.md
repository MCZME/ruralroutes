## 设计摘要

当前更倾向于把热带草原理解成：

- `边地过渡区`
- `粗放生产区`
- `手工业中转区`

热带草原回答的问题不是“平原换了金合欢木”，而是：

**“边地怎样把木材、牧产和土料先加工成能运输、能交换的货。”**

当前核心判断：

- 热带草原稳定输出金合欢木、木炭、皮货、牧产和陶瓦线货物。
- 热带草原长期需求石材、黏土、硬质配套、工具和少量工业输入。
- 它应作为加工中段，而不是成熟腹地或路线终点。

## 群系层

### 内容分层

| 类型 | 当前倾向 | 说明 |
|---|---|---|
| 基础资源 | 金合欢原木、生牛肉、皮革、羊毛、少量兔子皮 | 建立旱地木材和牧业底色 |
| 加工品 | 金合欢木板、木炭 | 表达边地会把本地资源粗加工成可运输货 |
| 群系记忆点货 | 黄色带釉陶瓦为主，兔子皮为辅 | 陶土 / 装饰线更能撑起记忆点 |
| 常备需求 | 石材 / 黏土类与硬质配套为主，矿物 / 金属为次 | 表达边地缺配套，而不是简单缺矿 |

### 设计边界

当前需要避免：

- 把热带草原写成“平原 + 金合欢木”。
- 把它写成“针叶林换木种”。
- 让矿物 / 金属缺口盖过石材、黏土和硬质配套这条主需求。

## 商路循环

### 流向定位

热带草原在商路里的主要职责是：

- 输出木材、木炭和牧产。
- 消费硬质配套、黏土、染料和工具。
- 把边地资源加工成陶瓦、带釉陶瓦和皮货。

当前低价流出货：

- 金合欢原木、金合欢木板、木炭
- 皮革、羊毛、兔子皮
- 生牛肉 / 熟牛肉
- 陶瓦、黄色带釉陶瓦

当前高价流入货：

- 石材、黏土
- 砂岩、玻璃等硬质配套
- 染料
- 铁锭、粗铁
- 斧头、剪刀等工具
- 少量口粮和燃料

### 推荐路线

当前热带草原最适合支撑两条循环：

1. `针叶林 / 热带草原木炭 -> 沙漠染坊 / 热带草原陶瓦 -> 装饰建材 -> 平原或其他建造节点`
2. `沙漠砂岩 / 玻璃 / 染料 -> 热带草原陶瓦 -> 黄色带釉陶瓦 -> 平原消费或建造需求`

推荐上游：

- 平原：工具、口粮、铁制配套和基础工业输入。
- 沙漠：砂岩、玻璃、染料和装饰材料。
- 针叶林：木炭、云杉木和冷地副产物。
- 积雪平原：骨块、皮革或寒地副产物。

推荐下游：

- 沙漠：消化木炭、木材和皮货，支撑染坊与采石。
- 积雪平原：消化木炭、木材、皮革和补给。
- 平原：消费带釉陶瓦、皮货、木炭和边地加工货。
- 建造型节点：消费陶瓦、染装货和装饰建材。

## 主题分工

### savanna_woodworks

当前身份：

- 边地木场
- 木炭加工点
- 金合欢木货集散点

当前偏向：

- 出售：金合欢原木、金合欢木板、木炭、木棍、少量简单木制品。
- 收购：斧头、铁锭、粗铁、少量工具材料。
- 代表货：鞍可作为极低库存旅途亮点，但不作为核心身份。
- 主题事件：木料放货、木炭出货、木料订单。
- 合约态度：不一定需要；若有，应围绕木材、木炭或收购型木料订单。

循环职责：

- 低价流出：金合欢木、金合欢木板、木炭。
- 高价流入：斧头、铁锭、粗铁、少量工具材料。
- 推荐下游：沙漠染坊、沙漠采石、积雪平原驿站、热带草原陶瓦。

### savanna_terracotta

当前身份：

- 陶土作坊
- 烧制窑场
- 装饰建材输出点

当前偏向：

- 出售：陶瓦、黄色带釉陶瓦、染料、装饰方块、窑烧类成品。
- 收购：黏土、木炭、金合欢木料、染料、玻璃、砂岩、少量口粮。
- 代表货：黄色带釉陶瓦。
- 主题事件：新窑开火、陶瓦出货、建材订单、装饰货热销。
- 合约态度：最自然可承接合约，例如陶瓦 / 黏土 / 燃料 / 染料相关加工或收购型窑场订单。

循环职责：

- 低价流出：陶瓦、黄色带釉陶瓦、窑烧装饰货。
- 高价流入：黏土、木炭、染料、玻璃、砂岩或其他硬质配套。
- 推荐下游：平原建设消费、沙漠装饰需求、后续建造型节点。

### savanna_herder

当前身份：

- 边地牧业
- 旷野皮货供应点
- 粗放牲口经济

当前偏向：

- 出售：皮革、羊毛、生牛肉 / 熟牛肉、兔子皮、少量皮货或粗放补给。
- 收购：剪刀、斧头、牧业工具、石材 / 金属工具、少量硬质配套。
- 代表货：兔子皮。
- 主题事件：皮货短缺、牧群迁徙、旱季整备、畜产放货。
- 合约态度：可以没有；若有，更适合皮货订单、肉货打包或副产物换货币。

循环职责：

- 低价流出：皮革、羊毛、兔子皮、肉货。
- 高价流入：剪刀、斧头、石材 / 金属工具、少量硬质配套。
- 推荐下游：积雪平原驻留、针叶林毛皮线、平原牧货消费。

## 设计约束

- `savanna_woodworks` 的鞍只作为弱代表货或旅途亮点，不能抢 `savanna_herder` 的牧业身份。
- `savanna_terracotta` 是热带草原最强视觉记忆点，但不能让整个群系都变成陶瓦主题。
- 热带草原需求继续强调“配套不足”，不要回落成泛泛缺矿。
