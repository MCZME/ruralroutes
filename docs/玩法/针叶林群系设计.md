## 设计摘要

当前更倾向于把针叶林理解成：

- `能生产的冷地`
- `林地采集区`
- `冷地生产中段`

针叶林回答的问题不是“这里很冷”，而是：

**“冷地里哪些地方还能靠森林、采集和边地手艺自己运转起来。”**

当前核心判断：

- 针叶林稳定输出云杉木、木炭、浆果、毛皮和部分驻留用品。
- 针叶林长期需求更好的工具、金属输入和生产升级材料。
- 它和积雪平原的区别是：针叶林缺升级能力，积雪平原缺驻留补给。

## 群系层

### 内容分层

| 类型 | 当前倾向 | 说明 |
|---|---|---|
| 基础资源 | 云杉原木、胡萝卜、甜菜根、马铃薯、皮革、羊毛 | 建立森林资源、冷地小农和毛皮副产物底色 |
| 加工品 | 云杉木板、木炭 | 表达林业初加工和边地生活 |
| 群系记忆点货 | 甜浆果为主，营火为辅 | 甜浆果更适合做针叶林共同记忆点 |
| 常备需求 | 更高阶工业输入为主，少量本地基础资源回收 | 缺的是生产效率，不是基础生活能力 |

### 设计边界

当前需要避免：

- 把针叶林写成普通木材区。
- 把针叶林写成热带草原换木种。
- 把针叶林写成积雪平原式补给终端。

## 商路循环

### 流向定位

针叶林在商路里的主要职责是：

- 输出木材、木炭、浆果和毛皮。
- 消费工具、金属和生产升级材料。
- 为雪地、沙漠和热带草原提供燃料与林产货。

当前低价流出货：

- 云杉原木、云杉木板、木炭
- 云杉船
- 甜浆果、发光浆果
- 皮革、羊毛
- 营火

当前高价流入货：

- 铁斧、石斧
- 铁剑、剪刀、弓、箭
- 铁锭、粗铁、煤炭
- 骨粉
- 少量能提高林业和采集效率的工具材料

### 推荐路线

当前针叶林最适合支撑两条循环：

1. `平原工坊工具 -> 针叶林 -> 云杉木 / 木炭 / 云杉船 -> 积雪平原或沙漠`
2. `平原骨粉 / 工具 -> 针叶林浆果 -> 积雪平原驿站或平原消费节点`

推荐上游：

- 平原工坊：石制工具、铁制工具和基础装备。
- 平原粮仓：少量口粮和骨粉链输入。
- 沙漠染坊：部分染料或加工材料。
- 热带草原：木炭、皮货和边地工具配套。

推荐下游：

- 积雪平原：消化云杉木、木炭、营火和皮货。
- 沙漠：消化木炭和木材，支撑染坊、采石和玻璃加工。
- 热带草原：消化木炭和部分工具材料。
- 平原：消费毛皮、浆果和木炭。

## 主题分工

### taiga_lumber

当前身份：

- 林场
- 木炭窑
- 林区河运和木货集散点

当前偏向：

- 出售：云杉原木、云杉木板、木炭、云杉船、少量林区实用品。
- 收购：斧头、铁锭、粗铁、煤炭等生产升级材料。
- 代表货：云杉船。
- 主题事件：伐木季、新木料放货、木炭窑开火、林区河运畅通。
- 机会合约：`云杉原木 + 木炭 -> 云杉船`。

循环职责：

- 低价流出：云杉原木、云杉木板、木炭、云杉船。
- 高价流入：斧头、铁锭、粗铁、煤炭。
- 推荐下游：积雪平原驿站、沙漠染坊、热带草原陶瓦。

### taiga_berries

当前身份：

- 林地采集点
- 冷地小农和浆果货摊
- 稀有浆果供应点

当前偏向：

- 出售：甜浆果、马铃薯、甜菜根、胡萝卜、发光浆果。
- 收购：骨粉、锄头、少量铁制工具或生产升级材料。
- 代表货：发光浆果。
- 主题事件：浆果季、林地采集旺季、发光浆果需求上升。
- 机会合约：`甜浆果 + 骨粉 -> 发光浆果`，语义更像林下培育成功的一批货。

循环职责：

- 低价流出：甜浆果、冷地作物、发光浆果。
- 高价流入：骨粉、锄头、少量铁制工具。
- 推荐下游：积雪平原补给点、平原消费节点、稀有食物需求事件。

### taiga_fur

当前身份：

- 林地猎户
- 毛皮和纤维货摊
- 林区驻留用品供应点

当前偏向：

- 出售：皮革、羊毛、兔肉或其他猎物类货物、营火、弓、箭。
- 收购：铁剑、铁斧、剪刀、木炭、少量毛皮处理材料。
- 代表货：营火，但语义应是林地驻留，不是雪原补给站。
- 主题事件：冬毛需求、毛皮短缺、猎户整备、入冬前交易。
- 机会合约：`皮革 + 木炭 -> 营火`。
- 战前补给：可作为弓箭、营火和皮革类补给的冷地来源，但不替代平原工坊的装备主线。

循环职责：

- 低价流出：皮革、羊毛、毛皮副产物、营火、弓、箭。
- 高价流入：铁剑、铁斧、剪刀、木炭。
- 推荐下游：积雪平原驿站、平原牧货消费、寒地驻留路线。

## 设计约束

- 甜浆果保留为群系记忆点，发光浆果作为 `taiga_berries` 的主题升级货。
- 营火在针叶林和积雪平原都能出现，但语义必须区分：针叶林是林地驻留，积雪平原是寒地补给。
- 针叶林需求继续围绕工具和升级能力，不写成单纯缺燃料或缺口粮。
