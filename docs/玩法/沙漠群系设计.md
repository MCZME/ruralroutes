## 设计摘要

当前更倾向于把沙漠理解成：

- `干旱建材区`
- `染装加工区`
- `外来补给消费区`

沙漠回答的问题不是“这里很热”，而是：

**“这片区域能把外来的木材、燃料和补给，转换成什么干旱建材与染装货。”**

当前核心判断：

- 沙漠稳定输出砂质材料、仙人掌、玻璃、染料和装饰加工货。
- 沙漠长期需求木材、燃料、水桶、口粮和少量工具 / 金属输入。
- 玩家不应靠原地挖沙完成主要循环，而应带外来缺口货进入沙漠，换走加工后的建材和染装货。

## 群系层

### 内容分层

| 类型 | 当前倾向 | 说明 |
|---|---|---|
| 基础资源 | 沙子、砂岩、仙人掌、少量兔子皮 / 生兔肉 | 建立“干旱材料区”的基础预期 |
| 加工品 | 玻璃、錾制砂岩 | 表达沙漠会把本地资源做成基础建材 |
| 群系记忆点货 | 绿色染料为主，高阶砂岩装饰为辅 | 染装线更适合成为沙漠共同记忆点 |
| 常备需求 | 木材 / 燃料为主，水桶为环境轴强化项，矿物 / 金属为弱缺口 | 需求重点来自干旱环境和加工成本 |

### 设计边界

当前需要避免：

- 把沙漠做成普通矿区或泛石材市场。
- 把绿洲写成沙漠版平原粮仓。
- 让所有主题都靠合约成立；沙漠主题可以没有合约。

## 商路循环

### 流向定位

沙漠在商路里的主要职责是：

- 消费木材、燃料和外来补给。
- 输出砂岩、玻璃、染料和装饰货。
- 把外来输入转化为更高辨识度的建材和视觉货。

当前低价流出货：

- 沙子、砂岩、红砂岩、錾制砂岩
- 玻璃
- 仙人掌
- 绿色染料、青色染料
- 陶瓦 / 装饰类成品

当前高价流入货：

- 原木、木炭、煤炭
- 面包或旅途口粮
- 水桶
- 骨粉、黏土、混凝土粉末
- 少量工具和金属输入

### 推荐路线

当前沙漠最适合支撑两条循环：

1. `平原 / 针叶林补给与燃料 -> 沙漠 -> 砂岩 / 玻璃 / 染料 -> 平原或热带草原`
2. `积雪平原冰雪货 -> 沙漠绿洲 / 染坊 -> 染料 / 绿洲补给 / 装饰货 -> 热带草原或平原`

推荐上游：

- 平原：口粮、基础工具、部分工业输入。
- 热带草原：金合欢木、木炭、皮货。
- 针叶林：云杉木、木炭、林业加工货。
- 积雪平原：冰、浮冰、蓝冰和冷藏叙事货。

推荐下游：

- 热带草原陶瓦线：承接染料、玻璃、建材和硬质配套。
- 平原腹地：消费玻璃、砂岩和染料。
- 后续建造型节点：承接沙漠建材与染装货。

## 主题分工

### desert_quarry

当前身份：

- 采砂场
- 砂岩工地
- 干旱建材输出点

当前偏向：

- 出售：沙子、砂岩、红砂岩、錾制砂岩、基础砂岩制品。
- 收购：木材、燃料、少量口粮、基础工具。
- 代表货：錾制砂岩或更有辨识度的砂岩装饰货。
- 主题事件：砌筑赶工、采坑放货、石料大单。
- 合约态度：可以没有；若有，应围绕砂材升级为砂岩制品。

循环职责：

- 低价流出：沙子、砂岩、红砂岩、砂岩制品。
- 高价流入：木材、燃料、口粮、基础工具。
- 推荐下游：热带草原陶瓦线、平原建设需求、其他建造节点。

### desert_oasis

当前身份：

- 干旱区生存缓冲点
- 商旅补给点
- 少量耐旱作物来源

当前偏向：

- 出售：仙人掌、少量作物、少量口粮、金西瓜、兔肉煲。
- 收购：水桶、冰雪货、口粮、骨粉、少量矿物 / 金属输入。
- 代表货：金西瓜、兔肉煲。
- 主题事件：绿洲收成、水边补给紧张、商队停靠压力。
- 合约态度：不一定需要，避免做成沙漠粮仓。

循环职责：

- 低价流出：少量耐旱补给、仙人掌、兔肉煲、金西瓜。
- 高价流入：水桶、冰雪货、口粮、骨粉。
- 推荐下游：沙漠内部加工主题、寒地或长途路线补给节点。

### desert_dyeworks

当前身份：

- 染料作坊
- 陶瓦和装饰成品作坊
- 沙漠视觉货加工点

当前偏向：

- 出售：绿色染料、青色染料、陶瓦、带釉陶瓦、玻璃或其他装饰加工货。
- 收购：仙人掌、骨粉、黏土球、沙子、木炭、混凝土粉末。
- 代表货：青色染料，或更高一级染装 / 陶瓦成品。
- 主题事件：染料热卖、新窑开火、装饰货出清。
- 合约态度：最适合承接作坊订单，可用货币结算，例如 `黏土 + 混凝土粉末 -> 铁币 + 铜板`。

循环职责：

- 低价流出：绿色染料、青色染料、陶瓦、装饰货。
- 高价流入：仙人掌、骨粉、黏土、木炭、混凝土粉末。
- 推荐下游：热带草原陶瓦、平原装饰消费、建造型需求节点。

## 设计约束

- 冰雪货进入沙漠时，以绿洲 / 染坊的高需求与事件窗口为主，不作为全沙漠稳定大宗需求。
- `desert_oasis` 保留“稀缺补给”语义，不能越写越像稳定农业主题。
- `desert_dyeworks` 的合约可以多做收购型订单，不必都输出物品。
