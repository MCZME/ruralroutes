## 设计摘要

当前更倾向于把平原理解成：

- `定居腹地`
- `稳定民生区`
- `商路启动区`

平原回答的问题不是“这里有什么稀有资源”，而是：

**“这片区域怎样稳定把人养活，并为商路提供第一批周转货。”**

当前核心判断：

- 平原常备供给应强在农产、口粮、牧产和基础工具。
- 平原常备需求应集中在工业输入、燃料和少量农资上。
- 平原是启动资金和补给来源，但不能变成什么都买、什么都卖的默认市场。
- 玩家从平原出发后，应自然被推向雪地、沙漠、针叶林或热带草原，而不是长期留在平原刷交易。

## 群系层

### 内容分层

| 类型 | 当前倾向 | 说明 |
|---|---|---|
| 基础资源 | 小麦、胡萝卜、马铃薯、生牛肉、生羊肉、皮革、羊毛、面包 | 建立“农业腹地 / 民生区”的第一层预期 |
| 加工品 | 少量石锄、石斧等低阶民生工具 | 只表达基础生产配套，不上升成完整工具市场 |
| 群系记忆点货 | 金胡萝卜为主，鞍作为弱候选 | 高级口粮线更适合作为平原共同记忆点 |
| 常备需求 | 矿物 / 金属输入为主，燃料为弱缺口，少量木头 / 石头回收 | 平原缺的是生产升级能力，不是活命资源 |

### 设计边界

当前需要避免：

- 把平原写成“什么都能收、什么都能卖”的默认中转站。
- 把铁制工具、铁锭、高级装备直接上升成群系共同底色。
- 让平原的民生供给宽度压掉其他群系的路线价值。

## 商路循环

### 流向定位

平原在商路里的主要职责是：

- 提供稳定启动货币的低门槛入口。
- 输出口粮、牧产和基础工具。
- 消费工业输入、燃料和农资。

当前低价流出货：

- 面包、烤马铃薯、南瓜派、苹果
- 南瓜、西瓜
- 皮革、羊毛、羽毛
- 石制工具、铁制工具
- 锁链装备、铁制装备
- 弓、箭等基础战前补给

当前高价流入货：

- 铁锭、粗铁
- 煤炭、木炭
- 少量木材、石料
- 骨粉、骨块
- 其他生产升级材料

### 推荐路线

当前平原最适合支撑两条循环：

1. `平原口粮 -> 积雪平原 -> 冰雪货 -> 沙漠 -> 染料 / 建材 -> 平原或热带草原`
2. `平原工具 -> 针叶林 / 积雪平原 / 热带草原 -> 木炭 / 骨块 / 皮货 -> 平原粮仓或沙漠染坊`

推荐上游：

- 针叶林：木炭、云杉木、林地副产物。
- 热带草原：金合欢木、木炭、皮货。
- 积雪平原：骨块、兔肉、皮革。
- 沙漠：玻璃、染料、砂岩装饰货。

推荐下游：

- 积雪平原：消化口粮、木材、燃料、工具。
- 沙漠：消化口粮、木材、燃料、水桶类补给。
- 针叶林：消化工具和工业输入。
- 热带草原：消化工具、硬质配套和部分工业输入。

## 主题分工

### plains_granary

当前身份：

- 粮食储备点
- 基础口粮输出点
- 平原“稳定养人”气质的核心主题

当前偏向：

- 出售：小麦、胡萝卜、马铃薯、南瓜、西瓜、甘蔗、苹果、面包、烤马铃薯、南瓜派、金胡萝卜倾向。
- 收购：少量本地农产、铁 / 煤 / 粗铁、骨粉、基础木石工具。
- 额外货物：堆肥桶、少量农具和储备型食物。
- 代表货：金胡萝卜、金西瓜。
- 主题事件：丰收、储粮紧张、口粮加工。
- 机会合约：`苹果 + 金粒 -> 金苹果`。

循环职责：

- 低价流出：口粮和基础农产。
- 高价流入：骨粉、燃料、少量贵金属输入。
- 推荐下游：积雪平原驿站、沙漠绿洲、长线商路补给点。

### plains_pasture

当前身份：

- 牲口经济
- 皮货和羊毛
- 轻微道路与远行感

当前偏向：

- 出售：皮革、羊毛、羽毛、生牛肉 / 熟牛肉、生羊肉、皮革装备。
- 收购：剪刀、木斧 / 石斧、木剑 / 石剑等牧场日常消耗工具。
- 额外货物：少量旅途、牲口照料和户外实用品。
- 代表货：拴绳、命名牌。
- 主题事件：牧棚满仓、皮货紧俏、牧群整备。
- 机会合约：`皮革 + 白羊毛 + 铁粒 -> 鞍`。
- 战前补给：可承接羽毛、皮革装备和床材料相关补给，但不承担武器主线。

循环职责：

- 低价流出：皮革、羊毛、羽毛、牲口管理用品。
- 高价流入：木制 / 石制武器工具、剪刀、少量五金配件。
- 推荐下游：寒地驻留点、猎户主题、需要皮革和纤维的边地主题。

### plains_workshop

当前身份：

- 平原低阶加工点
- 腹地生产配套
- 工具贸易起点

当前偏向：

- 出售：石制工具、铁制工具、锁链装备、铁制装备、弓、箭、少量木料和简单加工材料。
- 收购：铁锭、粗铁、煤 / 木炭、木板、少量工具生产材料。
- 额外货物：基础工具、简单耐用品、服务腹地生产的加工货。
- 代表货：钻石工具、附魔金质工具。
- 主题事件：工料短缺、农具赶工、工坊出货。
- 机会合约：工具材料转化为更高阶工具或铁砧类招牌货。
- 战前补给：平原最适合稳定承接弓、箭、基础护甲和工具准备。

循环职责：

- 低价流出：石制工具、铁制工具、锁链装备、铁制装备、弓、箭。
- 高价流入：铁锭、粗铁、煤炭、木炭、木板。
- 推荐下游：针叶林林业、积雪平原猎户、热带草原木作、沙漠采石。

## 设计约束

- 平原需求权重保持窄口径，尤其不能让它吞掉所有外来货。
- `plains_workshop` 的高级代表货保持“偶尔亮点”，不能表现成平原稳定量产高级装备。
- 平原的循环价值来自“出发”和“补给”，不是最终利润终点。
