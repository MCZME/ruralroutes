用于明确 [概述](../概述.md) 中“理论上仅靠本模组资源链可击杀末影龙”的具体边界。

本文件只讨论终局资源链是否成立；具体价格、库存档位、主题修正和事件幅度见 [价值与库存设计](../待办/价值与库存设计.md)。

## 核心结论

- 玩家可以只依靠村庄贸易系统获得通关所需关键材料。
- 交易系统只替代资源获取，不替代原版终局动作。
- `minecraft:ender_eye` 默认不直接售卖，玩家需要亲手合成。
- `minecraft:ender_pearl` 与 `minecraft:blaze_rod` 属于 T5 终局进口货。
- T5 终局进口货应高价、低库存、随机出现，不负责倒卖利润。
- 终局进口层保留三种入口：全局稀有货架、稀有进口事件、高价值路线收束合约。
- 单村综合上限建议：末影珍珠不超过 `3`，烈焰棒不超过 `2`。

## 通关边界

这里说的“仅靠本模组资源链可击杀末影龙”，指玩家获取关键资源时不必依赖原版关键掉落、矿洞或下界要塞。

但下列动作仍由玩家亲自完成：

| 动作 | 是否保留 | 原因 |
|---|---|---|
| 寻找村庄并建立商路 | 是 | 这是模组核心体验本身 |
| 合成末影之眼 | 是 | 保留材料凑齐后的仪式感 |
| 抛掷末影之眼寻找要塞 | 是 | 保留原版终局前的空间探索 |
| 激活末地传送门 | 是 | 保留通关门槛的实体操作 |
| 进入末地并击杀末影龙 | 是 | 不能被交易系统替代 |

不应作为前置阶段常规商品或能力提供：

- 已合成好的 `minecraft:ender_eye`
- 末地传送门框架或已激活传送门
- 要塞坐标、强制定位、直接传送到要塞
- `minecraft:dragon_breath`
- `minecraft:elytra`
- `minecraft:shulker_shell`
- 其他明显属于击龙后阶段的战利品

## 资源链结构

### 关键资源清单

| 类别 | 资源 | 商路必须提供 | 推荐载体 | 说明 |
|---|---|---|---|---|
| 基础补给 | 食物、木材、皮革、基础方块 | 是 | 常规卖品 | 支撑旅途与战前准备 |
| 战斗基础 | 弓、箭、铁制工具、常规防具替代品 | 是 | 常规卖品 / 工坊类合约 | 不要求顶配，但要能打 |
| 关键材料 | `minecraft:ender_pearl` | 是 | 全局稀有货架 / 进口事件 / 收束合约 | 不能缺失，否则无法开门 |
| 关键材料 | `minecraft:blaze_rod` | 是 | 全局稀有货架 / 进口事件 / 收束合约 | 优先提供烈焰棒，由玩家加工烈焰粉 |
| 定位与开门 | `minecraft:ender_eye` | 否 | 玩家合成 | 保留原版动作，不直接售卖 |
| 战斗强化 | 药水、附魔、高阶护甲 | 否 | 可选 | 不作为理论可通关硬要求 |

### 分阶段节奏

| 阶段 | 目标 | 商路作用 |
|---|---|---|
| 起步阶段 | 靠村庄活下来 | 提供基础食物、木材、简单建材、早期工具和最早一批货币 |
| 中期阶段 | 形成跨群系运输 | 提供跨村利润、高价值代表货和稳定补给 |
| 终局前阶段 | 取得维度材料 | 通过终局进口层逐步获得珍珠和烈焰棒 |
| 寻门阶段 | 玩家亲手完成最终准备 | 玩家合成末影之眼、寻找要塞、激活传送门 |
| 战斗阶段 | 靠贸易准备，但不靠贸易代打 | 提供食物、箭矢、床材料、方块和常规战斗工具 |

## 终局进口层

前往末地的关键材料不应只被理解成“所有主题小概率刷出的稀有商品”。

如果只这样处理，后期行为容易退化成：

`跑商赚钱 -> 到处刷村庄货架 -> 买末影珍珠 / 烈焰棒`

因此当前把关键材料放入“终局进口层”。它不是新手期普通商品，也不是某个群系的固定产业，而是玩家建立跨群系商路后逐步接触到的高价值异域货物流。

终局进口层位于内容结构的最后一层：

1. 群系共享稳定流通层
2. 主题强化与额外内容层
3. 全局随机稀有卖品层
4. 终局进口层

### 三种入口

| 入口 | 作用 | 说明 |
|---|---|---|
| 全局稀有货架 | 保底可遇见 | 所有主题都有小概率刷出，低库存、高价格 |
| 稀有进口事件 | 制造路线目标 | 短期提高关键材料或相关进口货出现率，让玩家愿意绕路 |
| 高价值路线收束合约 | 奖励完整跑商链 | 用高价值商路货和高面额货币换取终局材料 |

三者分工：

- 全局稀有货架负责不会锁死。
- 稀有进口事件负责有时值得改路线。
- 高价值路线收束合约负责把跑商成果转化为通关材料。

## 关键材料发放规则

| 资源 | 常态稀有货架 | 稀有进口事件 | 收束合约 | 单村综合上限 | 说明 |
|---|---:|---:|---:|---:|---|
| `minecraft:ender_pearl` | `0-1` | `1-3` | `1-2` | `3` | 所有主题都可能出现，但不可常驻批发 |
| `minecraft:blaze_rod` | `0-1` | `1-2` | `1-2` | `2` | 优先卖烈焰棒，让玩家自行合成烈焰粉 |
| `minecraft:ender_eye` | `0` | `0` | 默认不提供 | `0` | 保留玩家合成动作 |

设计口径：

- 末影珍珠和烈焰棒都不作为前中期普通大宗卖品。
- 两者都应依靠“全图都可能出现”避免锁档。
- 单村出现时库存很低，形成多村、多次、分批凑齐的节奏。
- 稀有事件可以制造短期窗口，但不是唯一来源。
- 收束合约必须吃进跨群系贸易成果，不吃本地采集物直接白给。

## 高价值路线收束合约

终局材料合约不应像普通合约那样用本地采集物直接兑换。

它更适合吃进：

- 高面额货币
- 高价值商路货
- 多群系路线才能自然获得的加工货

候选输入包括：

- `蓝冰`
- `黄色带釉陶瓦`
- `金苹果`
- 高价值工具或装备
- 高面额货币
- 其他由跨群系路线自然产出的高价值货

收束合约按主题分组出现，但不按主题垄断终局材料产出。

| 分组 | 适合主题 | 输入方向 | 输出方向 |
|---|---|---|---|
| 工坊金属线 | `plains_workshop` | 钻石工具、附魔金质工具、铁砧、金币 | 少量烈焰棒或终局进口货 |
| 装饰窑制线 | `desert_dyeworks`、`savanna_terracotta` | 青色染料、黄色带釉陶瓦、玻璃、陶瓦 | 少量末影珍珠或金币 |
| 寒地远货线 | `snowy_iceworks`、`snowy_waystation` | 蓝冰、营火、燃料、金币 | 少量终局进口货 |
| 农牧补给线 | `plains_granary`、`plains_pasture`、`desert_oasis` | 金苹果、金胡萝卜、皮革、鞍 | 高额货币或少量珍珠 |

这里的“按主题分组”只决定合约输入风味，不应让任何单一主题成为唯一终局入口。

## 物品池边界

现有 `#ruralroutes:pool/end_goods` 语义较宽，混合了通关前关键材料和击龙后战利品。

当前规则：

1. 前往末地前的关键材料用精确物品 ID 单独设计。
2. `minecraft:ender_pearl` 既属于 `#ruralroutes:pool/end_goods`，也可进入全局随机稀有池。
3. `minecraft:blaze_rod` 既属于 `#ruralroutes:pool/nether_goods`，也可进入全局随机稀有池。
4. `minecraft:shulker_shell`、`minecraft:elytra`、`minecraft:dragon_breath` 不进入前置通关链。

后续若要扩展击龙后的末地经济，再考虑把相关池细分为：

- 前往末地的关键材料
- 末地战利品

## 经济与随机性底线

| 底线 | 要求 |
|---|---|
| 不能锁档 | 玩家不能因为某个特定主题不刷出就永远缺关键材料 |
| 不能过早 | 新手刚接触贸易站时不应立刻买到珍珠或烈焰棒 |
| 不能单村解决 | 单个村庄或单个周期不应直接提供完整开门材料 |
| 不能廉价堆量 | 珍珠和烈焰材料应是阶段目标，不是普通消耗品价位 |
| 不能只剩刷商店 | 稀有事件和收束合约应把玩家拉回跑商路线 |

## 验收口径

以玩家准备 `18` 颗末影之眼为测试口径：

- `12` 颗用于开门
- `6` 颗用于寻路与损耗缓冲
- 等价需要 `18` 个 `minecraft:ender_pearl`
- 等价需要 `9` 个 `minecraft:blaze_rod`

路径成立性：

- 从新档开始，玩家理论上能通过持续跑商拿到上述材料。
- 不要求击杀末影人获取末影珍珠。
- 不要求进入下界要塞获取烈焰棒。

动作保留性：

- 玩家仍需合成末影之眼。
- 玩家仍需寻找要塞。
- 玩家仍需激活传送门。
- 玩家仍需亲自击杀末影龙。

节奏合理性：

- 不应在单村早期直接买齐。
- 不应因为纯 RNG 导致永远缺某一环。
- 应明显体现先建立商路，再冲击通关目标。

## 主题设计反推要求

后续 15 个主题至少需要满足：

1. 至少有一类主题承担稳定战前补给。
2. 至少有一类主题承担高价值进口合约。
3. 所有主题都应兼容全局稀有卖品入口。
4. 关键终局材料即便进入所有主题，也不应变成常驻货。
5. 若干主题能稳定承接弓、箭、方块、食物、床材料、基础护甲等战前补给。
6. 若干主题能承接高价值路线收束合约，把跑商成果转化为终局进口货。
