## 设计摘要

当前更倾向于把积雪平原理解成：

- `寒地终端`
- `远途停靠区`
- `高环境压力的特货输出点`

积雪平原回答的问题不是“这里有什么完整生产链”，而是：

**“哪些地方有寒地稀缺资源，但驻留和生存本身更依赖外来支持。”**

当前核心判断：

- 积雪平原稳定输出冰雪货、蓝冰线、兔肉、皮革和骨料。
- 积雪平原强需求口粮、木材、燃料、工具和驻留补给。
- 驿站出售补给不代表本地富产补给，而是商路带来的库存。

## 群系层

### 内容分层

| 类型 | 当前倾向 | 说明 |
|---|---|---|
| 基础资源 | 雪块、冰、生兔肉、皮革、少量马铃薯 / 甜菜根 | 建立寒地资源和狩猎生活底色 |
| 加工品 | 营火、浮冰 | 表达寒地驻留和冰雪初加工 |
| 群系特产 | 蓝冰为主，营火为辅 | 蓝冰更能成为玩家进雪地的理由 |
| 常备需求 | 木材 / 食物为主，工具 / 金属为次，少量冰雪和皮革回收 | 主缺口是生存轴和驻留成本 |

### 设计边界

当前需要避免：

- 把积雪平原写成雪地版针叶林。
- 把补给出售误解成本地富产。
- 让冰雪特产只变成玩家原地挖冰卖钱的单点交易。

## 商路循环

### 流向定位

积雪平原在商路里的主要职责是：

- 高价消费口粮、木材、燃料和工具。
- 输出冰、浮冰、蓝冰、骨块和寒地副产物。
- 作为远端终点制造“带补给进去、换特货出来”的路线感。

当前低价流出货：

- 雪块、冰、浮冰、蓝冰
- 兔肉
- 皮革
- 骨头、骨块
- 少量寒地副产物

当前高价流入货：

- 面包、烤马铃薯、南瓜派、苹果
- 原木、木炭、煤炭
- 营火
- 铁剑、铁斧、剪刀
- 弓、箭
- 少量硬通货式金属输入

### 推荐路线

当前积雪平原最适合支撑两条循环：

1. `平原口粮 + 针叶林木材 / 木炭 -> 积雪平原 -> 冰 / 浮冰 / 蓝冰 -> 沙漠`
2. `平原工坊工具 -> 积雪平原猎户 -> 骨块 / 皮革 / 兔肉 -> 平原粮仓或沙漠染坊`

推荐上游：

- 平原：口粮、基础补给、工具和腹地储备。
- 针叶林：云杉木、木炭、营火和毛皮相关物资。
- 热带草原：木炭、皮革、肉货和边地补给。
- 平原工坊：铁剑、铁斧、剪刀和猎户工具。

推荐下游：

- 沙漠绿洲：消化冰雪货，强化干旱区补给和冷藏叙事。
- 沙漠染坊：消化骨块 / 骨粉和冰雪货。
- 平原粮仓：消化骨块作为农业效率输入。
- 热带草原或建造节点：承接冰雪货和骨料的特殊需求。

## 主题分工

### snowy_waystation

当前身份：

- 远途驿站
- 寒地补给点
- 商路带来的外来物资库存

当前偏向：

- 出售：面包、少量寒地食物、原木、木炭、营火、少量可携带补给。
- 收购：口粮、木材、木炭、铁锭、生铁、金锭等周转和驻留资源。
- 主题特产：营火，语义是寒地临时整备和驿站补给包。
- 主题事件：口粮短缺、寒潮补给不足、旅人滞留、路线封冻前整备。
- 机会合约：`面包 + 云杉原木 -> 营火`，语义是临时整备，不是稳定工坊加工。

循环职责：

- 低价流出：营火、少量驿站库存、临时补给。
- 高价流入：口粮、木材、木炭、硬通货金属。
- 推荐下游：冰工、猎户、长线商路下一站。

### snowy_iceworks

当前身份：

- 冰窖
- 采冰场
- 高级冰货集散点

当前偏向：

- 出售：雪块、冰、浮冰、蓝冰。
- 收购：面包、熟食、木材、木炭、少量工具或硬质配套。
- 主题特产：蓝冰。
- 主题事件：冰库放货、融雪期、寒潮后冰货增多、蓝冰需求上升。
- 机会合约：`浮冰 + 木炭 -> 蓝冰`，语义是冰窖维护与运输整备，不按物理合成解释。

循环职责：

- 低价流出：雪块、冰、浮冰、蓝冰。
- 高价流入：面包、木炭、工具、采冰队补给。
- 推荐下游：沙漠绿洲、沙漠染坊、需要冷藏或冰道叙事的节点。

### snowy_hunter

当前身份：

- 寒地猎户
- 兔肉和皮货供应点
- 骨料与猎获副产物打包点

当前偏向：

- 出售：兔肉、皮革、羊毛 / 纤维、骨头、骨块、弓、箭。
- 收购：铁剑、铁斧、石斧、木炭、云杉原木、剪刀。
- 主题特产：骨块。
- 主题事件：猎获丰收、冬季皮货需求、猎具短缺、兽群迁移、骨料订单。
- 机会合约：`骨头 -> 骨块`，表达副产物打包运输。
- 战前补给：可补充弓箭和猎户用品，但应保持“远端补给不稳定、价格偏高”的感觉。

循环职责：

- 低价流出：兔肉、皮革、骨头、骨块、弓、箭。
- 高价流入：铁剑、铁斧、石斧、木炭、剪刀。
- 推荐下游：平原粮仓、沙漠染坊、寒地副产物需求事件。

## 设计约束

- `snowy_iceworks` 的蓝冰合约使用外来燃料或整备物资解释，避免像物理合成不成立的兑换。
- `snowy_waystation` 的补给库存始终解释成外来货，不能破坏雪地缺补给的群系判断。
- 积雪平原保持“窄而尖”，不要扩成完整冷地生产区。
