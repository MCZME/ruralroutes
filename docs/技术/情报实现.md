## 概述

情报是将内部技术数据（MarketEvent）转换为玩家可理解信息的中间层。

**情报定义**：
- 告诉玩家：**什么东西**在**哪里**发生了**什么变化**
- 不显示具体数值，只显示方向
- 生命周期与 MarketState 一致，实时生成，不持久化

---

## 数据结构

### RumorEntry

```java
public record RumorEntry(
    String targetNameKey,      // 物品或标签的翻译键
    MarketScopeType scopeType, // 作用域类型
    String scopeTargetName,    // 作用域目标名称
    String directionKey        // "up" | "down"
)
```

**字段说明**：

| 字段 | 来源 | 示例 |
|------|------|------|
| `targetNameKey` | MarketEvent.targetRef | `"item.minecraft.wheat"` 或 `"ruralroutes.tag.minecraft.planks"` |
| `scopeType` | MarketEvent.scopeType | GLOBAL / BIOME / THEME |
| `scopeTargetName` | MarketEvent.scopeTarget | 群系/主题翻译键，全局时为空 |
| `directionKey` | MarketEvent.getDirectionKey() | "up" 或 "down" |

## 情报生成

### RumorGenerator

```java
public static List<RumorEntry> generateFromMarketState(MarketState state, Random random)
```

**流程**：

1. 检查 MarketState 是否为空
2. 空状态 → 返回闲谈情报
3. 非空状态 → 遍历 events，逐条转换为 RumorEntry

### 目标名称解析

```java
private static String resolveTargetNameKey(MarketEvent event)
```

| targetRef 类型 | 翻译键格式 | 示例 |
|----------------|-----------|------|
| 物品 ID | `item.{namespace}.{path}` | `"item.minecraft.wheat"` |
| 标签（#开头） | `ruralroutes.tag.{namespace}.{path}` | `"ruralroutes.tag.minecraft.planks"` |

### 作用域目标解析

| scopeType | 翻译键格式 | 示例 |
|-----------|-----------|------|
| GLOBAL | 空 | - |
| BIOME | `biome.{namespace}.{path}` | `"biome.minecraft.plains"` |
| THEME | `ruralroutes.theme.{path}` | `"ruralroutes.theme.plains_granary"` |

## 文本模板

### 参数化模板

**参数占位符**：
- `%s` (arg0) = 物品/标签名称
- `%s` (arg1) = 作用域描述
- `%s` (arg2) = 涨跌描述

**模板列表**：

| 翻译键 | 中文内容 |
|--------|----------|
| `rumor.template.1` | %s在%s价格%s |
| `rumor.template.2` | %s的%s价格%s |
| `rumor.template.3` | 听说%s的%s价格%s |
| `rumor.template.4` | %s价格%s，%s受影响 |
| `rumor.template.5` | 商路传闻：%s的%s价格%s |

### 例句

- "小麦在世界各地价格上涨。"
- "木板类物品在平原一带价格下跌。"
- "商路传闻：平原粮仓的面包价格上涨。"

## 翻译键

### 涨跌描述

| 翻译键 | 中文 | 英文 |
|--------|------|------|
| `rumor.direction.up` | 上涨 | risen |
| `rumor.direction.down` | 下跌 | fallen |

### 作用域描述

| 翻译键 | 中文 | 英文 |
|--------|------|------|
| `rumor.scope.global` | 世界各地 | everywhere |
| `rumor.scope.biome` | %s一带 | %s area |
| `rumor.scope.theme` | %s | %s |

### 闲谈文本

| 翻译键 | 中文 |
|--------|------|
| `rumor.gossip.1` | 最近风平平静，没有特别的消息。 |
| `rumor.gossip.2` | 商路上没什么新鲜事。 |
| `rumor.gossip.3` | 各村庄物价平稳。 |

## 标签翻译

标签使用模组前缀的翻译键格式，需要在语言文件中定义：

| 翻译键 | 中文 | 英文 |
|--------|------|------|
| `ruralroutes.tag.minecraft.planks` | 木板类物品 | Plank items |
| `ruralroutes.tag.minecraft.crops` | 农作物类 | Crop items |
