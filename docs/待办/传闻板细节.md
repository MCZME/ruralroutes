## 情报结构

每条情报由三段拼接：

> **[物品池名称] + [波动方向描述] + [作用域暗示]**

## 文本库

### 波动方向描述

| 方向 | 可选文本 |
| :--- | :--- |
| 上涨 | 需求大增，收购价普遍上涨 |
| 上涨 | 最近很抢手，价格走高 |
| 上涨 | 供不应求，出手要趁早 |
| 上涨 | 许多人都在找，换起来更划算 |
| 下跌 | 供应过剩，收购价走低 |
| 下跌 | 最近不太好换，价格下跌 |
| 下跌 | 市场上太多了，暂时不值钱 |
| 下跌 | 需求减少，换的时候多看看价 |

### 作用域暗示

| 作用域 | 可选文本 |
| :--- | :--- |
| 全局 | 全图各村庄都…… / 各地商人都在…… / 据说到处都…… |
| 区域 | {群系名}的村庄…… / {群系名}一带…… / 往{群系名}方向的商人…… |

### 闲谈文本（无波动时）

- 最近风平浪静，没有特别的消息。
- 商路上没什么新鲜事，村民们各自忙碌着。
- 各村庄物价平稳，是个休息的好日子。

## 生成流程

1. 市场波动生成：确定全局和区域波动的物品池、群系、方向
2. 全局情报：池名 + 方向描述（随机选） + 全局作用域（随机选）
3. 区域情报：池名 + 方向描述（随机选） + 区域作用域（随机选，填入群系名）

## 示例

波动：全局木材上涨，区域沙漠矿物下跌

> 木材最近很抢手，各地商人都在高价收购。
>
> 矿物在沙漠一带供应过剩，换起来不太划算。

## 配置与本地化

- 文本库通过配置文件驱动，允许整合包作者修改
- 按 MC 语言文件方式支持多语言

## 已确定方向

- 情报层级：1条全局情报 + 1条区域情报
- 情报真实性：反映真实市场波动
- 交互方式：右键查看详细，表面显示简略图标
