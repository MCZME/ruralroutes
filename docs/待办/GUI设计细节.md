## 已确定方向

- 贸易站界面：出售/收购分区域，网格卡片布局
- 展示柜界面：独立小界面，只列特产和特殊交换规则
- 组件化设计：各区域拆分为独立 Widget 组件

## 贸易站 GUI 布局


### 尺寸规范

| 参数 | 值 | 说明 |
|------|-----|------|
| GUI 总宽度 | 320px | 略大于原版箱子 GUI |
| GUI 总高度 | 240px | 适配三个区域 |
| 卡片大小 | 18×18px | 与原版物品槽位一致 |
| 卡片间距 | 2px | |
| 区域间距 | 8px | 各区域间隔 |
| 边距 | 10px | |

### 组件拆分

| 组件 | 文件 | 职责 |
|------|------|------|
| `SellSectionWidget` | 出售清单组件 | 标题 + 9×2 网格卡片 |
| `BuySectionWidget` | 收购清单组件 | 标题 + 9×2 网格卡片 |
| `TradeAreaWidget` | 交易区组件 | 你付出/你获得 + 价值状态 + 确认按钮 |
| `CoinExchangeWidget` | 铸币快捷操作组件 | 四个货币转换按钮 |

### 颜色规范

| 区域 | 颜色 (ARGB) | 说明 |
|------|-------------|------|
| 出售清单卡片 | 0x4055FF55 | 半透明绿色 |
| 收购清单卡片 | 0x40FF5555 | 半透明红色 |
| 交易区-付出 | 0x40FFAA00 | 半透明橙色 |
| 交易区-获得 | 0x4000AAFF | 半透明蓝色 |
