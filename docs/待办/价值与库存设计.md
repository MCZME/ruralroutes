本文件用于记录乡野商路的价值层级、库存层级和跑商利润原则。它服从：

- [主题设计方案](../玩法/主题设计方案.md)
- [末影龙通关资源链方案](../玩法/末影龙通关资源链方案.md)

## 设计摘要

价值和库存不是最后填的数值，而是商路循环的骨架。

当前核心判断：

- 价格决定玩家为什么跑远路。
- 库存决定玩家不能在单村一次解决全部目标。
- 同地买卖不能套利。
- 顺路线跨群系应有明确利润。
- 高价值货要贵、少、分散。
- 终局材料是财富回收口，不是倒卖赚钱货。

一句话规则：

**T1-T4 是赚钱层，T5 是花钱层。**

## 价值层级

当前先按 5 个价值层讨论物品。

| 层级 | 定位 | 例子 | 库存倾向 |
|---|---|---|---|
| T1 基础消耗 | 活下来、跑起来、低级原料 | 面包、木头、沙子、基础作物 | 高库存 |
| T2 跑商基础货 | 能稳定跨村卖差价 | 木炭、玻璃、皮革、羊毛、基础工具 | 中库存 |
| T3 地方特产 | 有明显区域记忆点 | 金胡萝卜、青色染料、陶瓦、浮冰、云杉船 | 中低库存 |
| T4 高价值路线货 | 需要多段商路或主题强化 | 蓝冰、黄色带釉陶瓦、金苹果、钻石工具、附魔金质工具 | 低库存 |
| T5 终局进口货 | 击败末影龙前的关键材料 | 末影珍珠、烈焰棒 | 极低库存 |

基础价值区间按路线角色确定，允许相邻层级重叠：

| 层级 | 建议基础价值 | 说明 |
|---|---:|---|
| T1 基础消耗 | `1-5` | 启动、补给、低阶原料 |
| T2 跑商基础货 | `4-20` | 稳定跨村利润货 |
| T3 地方特产 | `6-40` | 有主题记忆点，可有少数高价特产 |
| T4 高价值路线货 | `24-120` | 多段路线成果和高价值合约输入 |
| T5 终局进口货 | `80-150` | 金币级财富回收，推进通关 |

这里的层级不是只看原版稀有度。

同一个物品应看它在商路中的角色：

- 是否支撑生存
- 是否支撑稳定跑商
- 是否代表地方记忆点
- 是否需要多段路线才能自然获得
- 是否用于终局通关而非赚钱

## 库存原则

库存按用途控制，而不是只按稀有度控制。

项目支持出售库存和收购容量分开设置，因此库存不应只讨论“高低”，而应拆成两个维度：

- 出售库存：村庄本期有多少现货可以卖给玩家，决定玩家能不能在这里进货。
- 收购容量：村庄本期还能吃进多少货，决定玩家能不能把路线货卖到这里。

这两个维度共同决定贸易角色：

| 贸易角色 | 出售库存 | 收购容量 | 作用 |
|---|---:|---:|---|
| 本地富产货 | 高 | 低 / 中 | 玩家在这里进货，运到别处卖 |
| 本地需求货 | 低 / 无 | 高 | 玩家把别处货运来赚钱 |
| 集散补给货 | 中 | 中 | 保持路线流动，不制造暴利 |
| 主题特产 | 低 | 低 / 中 | 有记忆点，但不能批发 |
| 加工工坊货 | 中低 | 高输入 | 村庄卖成品、吃原料 |
| 终局进口货 | 极低 | 极低 / 无 | 花钱目标，不是贸易货 |

### 出售库存档位

| 档位 | 建议范围 | 用途 |
|---|---:|---|
| SS0 极稀有现货 | `0-1` 或 `1-1` | 稀有进口、偶发货 |
| SS1 稀有现货 | `1-2` | 蓝冰、鞍、铁砧、金苹果等 |
| SS2 少量现货 | `2-4` | 高级特产、小批量加工品 |
| SS3 中低现货 | `4-8` | 工具、较贵食品、普通特产 |
| SS4 常规现货 | `8-16` | 主循环跑商货 |
| SS5 大宗现货 | `16-32+` | 大宗基础货、补给、低阶原料 |

### 收购容量档位

| 档位 | 建议范围 | 用途 |
|---|---:|---|
| BS0 极低需求 | `0-2` | 不鼓励玩家卖入 |
| BS1 少量回收 | `4-8` | 顺手回收，不作为路线终点 |
| BS2 常规需求 | `8-16` | 普通主题需求 |
| BS3 明确需求 | `16-32` | 明确路线终点 |
| BS4 大宗消化 | `32-48+` | 大宗吞吐口，适合基础货或事件需求 |

示例口径：

- 平原粮仓对作物可用 `SS5 / BS1`。
- 积雪驿站对食物和燃料可用 `SS2-SS3 / BS3`。
- 沙漠染坊对黏土、骨粉、混凝土粉末可用 `SS0-SS1 / BS3`。
- 针叶林木材可用 `SS5 / BS1`。
- 沙漠对木炭可用 `SS1 / BS3`。

价格告诉玩家哪里赚钱，库存告诉玩家哪里真的需要货。

### T1 基础消耗

目标：

- 保证玩家能跑起来。
- 保证新手不会因为补给断裂而被迫回到原版采集。

库存倾向：

- 高库存
- 价格低
- 跨区利润小

代表：

- 基础食物
- 基础木材
- 沙子、基础作物
- 少量低阶原料

### T2 跑商基础货

目标：

- 支撑最稳定的跨群系利润。
- 让玩家形成路线记忆。

库存倾向：

- 中库存
- 利润中等
- 适合作为主循环货

代表：

- 木炭
- 玻璃
- 皮革、羊毛
- 基础工具

### T3 地方特产

目标：

- 强化群系或主题记忆点。
- 让玩家愿意专门去某类村庄补货。

库存倾向：

- 中低库存
- 利润较高
- 不应稳定大量铺货

代表：

- 金胡萝卜
- 青色染料
- 陶瓦
- 浮冰
- 云杉船

### T4 高价值路线货

目标：

- 浓缩多段商路的收益。
- 成为高价值合约和终局进口层的输入候选。

库存倾向：

- 低库存
- 高价格
- 高利润
- 不可单村大量解决

代表：

- 蓝冰
- 黄色带釉陶瓦
- 金苹果
- 钻石工具
- 附魔金质工具

### T5 终局进口货

目标：

- 支撑仅靠商路击败末影龙。
- 消耗玩家财富和高价值路线货。
- 不负责赚钱。

库存倾向：

- 极低库存
- 极高价格
- 不适合作为倒卖货
- 单村只提供很少数量

代表：

- 末影珍珠
- 烈焰棒

## 价差原则

价格不只由物品稀有度决定，也由路线角色决定。

当前规则：

- 产地便宜。
- 需求地贵。
- 同地买卖不能套利。
- 顺路线跨群系可以有明确利润。
- 两段以上路线利润更高，但库存更低。
- T5 终局货不应成为利润来源。

典型方向：

- 平原卖口粮便宜，积雪平原收口粮贵。
- 针叶林卖木炭便宜，沙漠和积雪平原收木炭贵。
- 积雪平原卖冰雪货便宜，沙漠绿洲或染坊收冰雪货贵。
- 沙漠卖染料、玻璃和砂岩便宜，热带草原陶瓦线或平原建设需求收得更贵。

这里要避免：

- 同一村庄买入再卖出获利。
- 基础货利润过高，压掉特产和路线货。
- 高价值货库存过高，导致玩家不需要移动。
- 终局货可以倒卖获利。

## 主题修正区间

主题修正只负责常态流向，市场事件负责临时机会。

实现中最终价格会取整数，因此低基础价值物品不能依赖细微倍率表达价差。例如基础价值为 `1` 的物品，`1 × 1.01` 取整后仍然是 `1` 铜板。

### 取整影响

| 基础价值 | 有效修正口径 | 说明 |
|---:|---|---|
| `1` | 基本不要微调 | 需要很大倍率才会产生 1 铜板变化 |
| `2-3` | 用强档位 | `1.04`、`1.08` 这类微调多数无体感 |
| `4-8` | 可用中等档位 | 大约 `0.85-1.20` 开始明显 |
| `10-16` | 可用常规档位 | `0.90-1.15` 已有体感 |
| `20+` | 可用细档位 | 高价值货不宜过度放大 |

### 按价值段选择倍率

| 价值段 | 富产地 sell / buy | 需求地 sell / buy | 设计口径 |
|---|---:|---:|---|
| 低价值 `1-3` | `0.60-0.80 / 0.50-0.75` | `1.25-1.60 / 1.30-1.70` | 少用微调，主要靠大倍率和库存吞吐 |
| 中价值 `4-12` | `0.75-0.90 / 0.70-0.85` | `1.10-1.25 / 1.15-1.35` | 主循环跑商货的主要区间 |
| 高价值 `16+` | `0.85-0.96 / 0.80-0.92` | `1.04-1.16 / 1.08-1.22` | 避免高价值货因倍率过大变成暴利 |
| 终局进口 | `1.25-1.50 / 0.50-0.80` | 不作为需求地 | 消耗财富，不负责倒卖利润 |

字段含义：

- `sell` 是村庄卖给玩家的价格因子。
- `buy` 是村庄从玩家收购的价格因子。

常态主题修正原则：

- 富产地应 `sell` 低，玩家适合进货。
- 需求地应 `buy` 高，玩家适合卖入。
- 低价值物品需要更粗的倍率，或主要通过高库存 / 高收购容量体现流向。
- 高价值物品应使用更细的倍率，避免市场事件叠加后利润失控。

## 利润结构

当前更适合按层级分配利润预期。

| 层级 | 利润预期 | 设计目的 |
|---|---|---|
| T1 | 小利润 | 启动、补给、维持旅行 |
| T2 | 中等利润 | 主循环跑商货 |
| T3 | 较高利润 | 强化地方路线记忆 |
| T4 | 高利润 | 奖励多段路线和低库存货 |
| T5 | 不负责利润 | 消耗财富，推进通关 |

玩家理想成长节奏：

1. 用 T1 起步。
2. 用 T2 建立稳定路线。
3. 用 T3 和 T4 扩大利润。
4. 用积累的货币和高价值货换取 T5。
5. 把 T5 投入末影龙通关，而不是拿去倒卖。

## 当前基础价值落点

本节只记录内容取值，不重新定义价格公式和库存机制。

| 物品 | 基础价值 | 路线定位 |
|---|---:|---|
| 青色染料 | 6 | T3 地方特产，略高于普通染料池 |
| 黄色带釉陶瓦 | 16 | T4 装饰路线货，适合承接沙漠和热带草原路线 |
| 蓝冰 | 24 | T4 冰路线货，适合作为远距高价值货 |
| 末影珍珠 | 80 | T5 终局进口货，不负责倒卖利润 |
| 烈焰棒 | 100 | T5 终局进口货，接近金币级财富回收 |

## 货币面额配合

现有面额 `铜板 = 1`、`铁币 = 10`、`金币 = 100` 已能覆盖 T1-T5，不需要新增更高面额。

| 层级 | 货币体验 |
|---|---|
| T1 | 主要用铜板，少量铁币 |
| T2 | 铜板和铁币混用 |
| T3 | 逐渐以铁币计价 |
| T4 | 高段开始触碰金币，形成“赚大钱”的感觉 |
| T5 | 明确以金币为心理单位，铁币负责找零 |

当前 `末影珍珠 = 80`、`烈焰棒 = 100` 的基础价值适合承接金币级终局消耗。叠加终局进口的高价 `sell` 后，末影珍珠可自然接近金币单位，烈焰棒可进入 `120-150` 的阶段目标价格。

## 终局材料约束

`末影珍珠` 和 `烈焰棒` 属于 T5。

当前规则：

- 它们不是普通跑商利润货。
- 它们不应进入群系常备供给。
- 它们不应由单个主题永久独占。
- 它们可以通过全局稀有货架出现。
- 它们可以通过稀有进口事件出现。
- 它们可以通过高价值路线收束合约获得。

库存和价格应体现：

- 单村不可一次买齐。
- 单周期不可轻松解决。
- 多村、多次、多路线逐步凑齐。
- 玩家应明显感到这是阶段目标。

T5 单村库存上限：

| 来源 | 末影珍珠 | 烈焰棒 | 说明 |
|---|---:|---:|---|
| 常态稀有货架 | `0-1` | `0-1` | 遇见是惊喜，不作为稳定来源 |
| 稀有进口事件 | `1-3` | `1-2` | 事件期可以明显更有价值 |
| 高价值收束合约 | `1-2` | `1-2` | 吃 T4 路线货，不应白给 |
| 单村综合上限 | `3` | `2` | 防止一村解决太多终局进度 |

## 与事件的关系

事件不只改价格，也要改变路线吸引力。

当前建议：

- `release` 事件主要提高买入价值，让玩家适合在当地进货。
- `shortage` 事件主要提高卖出价值，让玩家适合运入补给。
- `demand` 事件主要制造高价出售窗口。
- 稀有进口事件主要服务 T5 和部分 T4，让玩家愿意绕路。

事件幅度和主题修正类似，但它会叠加，并且只持续一个贸易周期。

默认配置中：

- `market.max_delta = 0.50`，价格事件叠加后最多影响 ±50%。
- `market.max_stock_delta = 0.75`，库存事件叠加后最多影响 ±75%。

因此单条事件不应频繁接近上限。否则多事件叠加后会快速撞顶，事件之间失去差异。

### 事件幅度档位

| 事件层级 | 价格 `delta` | 库存 `stock` | 作用 |
|---|---:|---:|---|
| 普通全局事件 | `±0.10-0.18` | `±0.10-0.30` | 世界背景行情 |
| 群系 / 主题事件 | `±0.18-0.28` | `±0.20-0.45` | 明确跑商窗口 |
| 稀有事件 | `±0.30-0.45` | `±0.30-0.60` | 明显改路线的机会 |

### 按价值段选择事件幅度

| 价值段 | 价格事件 | 库存事件 | 设计口径 |
|---|---:|---:|---|
| 低价值 `1-3` | `±0.25` 起步 | `±0.35-0.60` | 小价格波动会被取整吞掉，主要靠吞吐变化 |
| 中价值 `4-12` | `±0.15-0.30` | `±0.20-0.45` | 主循环跑商货的主要事件区间 |
| 高价值 `16+` | `±0.08-0.20` | `±0.10-0.30` | 避免高价值货被事件放大成暴利 |
| 终局进口 | 可高价，但不做倒卖利润 | 极低增量 | 服务阶段目标，不服务跑商利润 |

### 事件类型口径

| 事件类型 | `delta` | `stock.sell` | `stock.buy` | 含义 |
|---|---:|---:|---:|---|
| 放货 / 丰收 | 负 | 正 | 负或小负 | 当地货多，适合进货 |
| 短缺 | 正 | 负 | 小正或中正 | 当地缺货，适合运入 |
| 需求 / 热销 | 正 | 小负或不变 | 正 | 村庄愿意高价收 |
| 滞销 / 过剩 | 负 | 正 | 负 | 便宜但不太收 |
| 稀有进口 | 正或高价 | 小正 | 低或无 | 出现少量高价值货 |

库存联动方向：

- 放货事件：卖价下降，出售库存上升。
- 短缺事件：买价上升，出售库存下降。
- 需求事件：买价上升，收购容量上升。
- 稀有进口事件：少量高价值货出现，但不应大批量铺货。

价格事件负责“赚不赚钱”，库存事件负责“这趟能不能跑成规模”。

## 与合约的关系

合约不应成为稳定批发渠道。

当前建议：

- T1 / T2 合约可以服务基础加工或打包。
- T3 合约可以强化地方主题。
- T4 合约可以收束多段路线。
- T5 合约必须吃进高价值路线成果，不应吃本地采集物。

高价值路线收束合约的输入方向：

- 高面额货币
- 蓝冰
- 黄色带釉陶瓦
- 金苹果
- 钻石工具或附魔金质工具
- 其他由多群系路线自然产出的高价值货

收束合约按主题分组出现，但不按主题垄断 T5 产出。

| 分组 | 适合主题 | 输入方向 | 输出方向 |
|---|---|---|---|
| 工坊金属线 | `plains_workshop` | 钻石工具、附魔金质工具、铁砧、金币 | 少量烈焰棒或终局进口货 |
| 装饰窑制线 | `desert_dyeworks`、`savanna_terracotta` | 青色染料、黄色带釉陶瓦、玻璃、陶瓦 | 少量末影珍珠或金币 |
| 寒地远货线 | `snowy_iceworks`、`snowy_waystation` | 蓝冰、营火、燃料、金币 | 少量终局进口货 |
| 农牧补给线 | `plains_granary`、`plains_pasture`、`desert_oasis` | 金苹果、金胡萝卜、皮革、鞍 | 高额货币或少量珍珠 |

这里的“按主题分组”只决定合约输入风味，不应让任何单一主题成为唯一终局入口。
